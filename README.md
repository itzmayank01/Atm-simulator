# ATM Simulation Website

A simple, modular ATM simulation web app using HTML, CSS, JavaScript, and Python (Flask).

## Features
- User login with secure PIN authentication
- Balance inquiry
- Cash withdrawal and deposit
- Transaction history

## Project Structure
- `frontend/` — HTML, CSS, JS for UI
- `backend/` — Flask app for API and logic

## Setup
1. Install Python 3 and Flask
2. Run the backend: `cd backend && flask run`
3. Open `frontend/index.html` in your browser
